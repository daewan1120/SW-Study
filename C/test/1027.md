# 1027. 포물선 최대 높이 계산

## 설명
초기 속도 v(m/s)와 발사 각도 θ(도)를 입력받아 포물선의 최대 높이 H를 계산하는 프로그램을 작성하세요.

- 공식
```
    H = (v² * sin²θ) / (2 * g)
    g = 9.8 m/s²
```
- θ는 입력으로 도 단위, 프로그램에서 라디안 변환 필요
  (rad = θ * π / 180)
- 소수점 3자리까지 출력

---

### 입력 예시 1
    20 45

### 출력 예시 1
    10.204

### 입력 예시 2
    30 30

### 출력 예시 2
    11.475

### 입력 예시 3
    10 60

### 출력 예시 3
    3.827
