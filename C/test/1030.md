# 1030. 택배 배달 요금 계산

## 설명
택배의 무게(kg)와 거리(km)를 입력받아 요금을 계산하고 출력하는 프로그램을 작성하세요.  

**요금 기준:**  
- 기본 요금: 5000원
- 무게 추가 요금:
   - 5kg 이하: 추가 없음
   - 5kg 초과 20kg 이하: 2000원 추가
   - 20kg 초과: 5000원 추가
- 거리 추가 요금:
   - 10km 이하: 추가 없음
   - 10km 초과: km당 100원 추가

### 입력 예시 1
    3 8

### 출력 예시 1
    5000원

### 입력 예시 2
    10 15

### 출력 예시 2
    7000원

### 입력 예시 3
    25 20

### 출력 예시 3
    9000원
