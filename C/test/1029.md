# 1029. 화폐 변환

## 설명
사용자로부터 한국 원(KRW) 금액을 입력받아  
다음 단위로 변환하는 프로그램을 작성하세요.

1 USD = 1463 KRW  
1 USD = 0.86 EUR  
1 USD = 156.1 JPY  

소수점 2자리까지 출력하세요.

### 입력 예시 1
    146300

### 출력 예시 1
    100.00
    86.00
    15610.00

### 입력 예시 2
    366451

### 출력 예시 2
    250.50
    215.43
    39055.05

### 입력 예시 3
    0

### 출력 예시 3
    0.00
    0.00
    0.00
