## 1033. 아스키 문자열 변환기

### **설명**
1자리 정수 키 N(0 < N < 6)와 알파벳 대문자로만 구성된 문자열을 입력받아 다음 규칙에 따라 문자열의 각 문자를 변환하여 출력하는 프로그램을 작성하세요.

변환 규칙은 각 문자의 아스키 코드 값의 홀짝 여부와 키 N을 함께 사용하여 결정됩니다.

```
- 홀수 아스키 코드: 문자의 아스키 코드 값이 홀수이면, 키 N을 더하세요.
- 짝수 아스키 코드: 문자의 아스키 코드 값이 짝수이면, 키 N을 빼세요.
```

변환 결과가 알파벳 범위('A'~'Z')를 벗어날 경우, 알파벳이 순환하도록 처리하세요.

```
- 'Z'를 넘어갈 경우 ('A' 쪽으로 다시 시작).
- 'A'보다 작아질 경우 ('Z' 쪽으로 다시 시작).
```

---

### **입력 및 출력 예시**

#### **입력 예시 1 (순환 발생)**
    5 B A D Z

#### **출력 예시 1**
    W F Y U

#### **입력 예시 2 (순환 발생)**
    3 X Y Z

#### **출력 예시 2**
    U B W
