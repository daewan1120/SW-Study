# Chapter 06 - C언어의 반복문 (Loops)

## 반복문이란?

- **동일한 코드를 여러 번 반복** 실행하기 위해 사용하는 구문
- 조건이 참인 동안 **블록 내부 코드가 반복 실행됨**
- C언어에서 제공하는 반복문 종류:
    - `for`
    - `while`
    - `do while`

---

## for문

- 반복 횟수가 **명확할 때** 사용
- **초기식 → 조건식 검사 → 블록 실행 → 증감식 → 반복**

```c
for (초기식; 조건식; 증감식) {
    // 반복할 코드
}
```

ex)

```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d번째 반복입니다.\n", i);
    }
    return 0;
}
```
출력:

    1번째 반복입니다.
    2번째 반복입니다.
    3번째 반복입니다.
    4번째 반복입니다.
    5번째 반복입니다.

---

## while문

- 조건을 먼저 검사하고, 참일 때만 반복 실행
- 반복 횟수가 정해지지 않았거나, 사용자 입력에 따라 달라질 때 사용

```c
while (조건식) {
    // 조건이 참이면 반복
}
```
ex)

```c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
```

출력:

    1
    2
    3
    4
    5

---

## do while문

- 조건과 상관없이 무조건 한 번은 실행
- 그런 다음 조건을 검사하여 반복 여부 결정

```c
do {
    // 최소 1회 실행
} while (조건식);
```

ex)

```c
int i = 1;

do {
    printf("%d\n", i);
    i++;
} while (i <= 5);
```

출력:

    1
    2
    3
    4
    5

## break와 continue
| 키워드 | 설명 |
| - | - |
| `break` | 반복문을 **즉시 종료** |
| `continue` | 아래 코드를 건너뛰고 **다음 반복**으로 진행 |

ex) 

```c
// break
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    printf("%d ", i);
}
```
출력:

    1 2 3 4

ex) 

```c
//continue
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    printf("%d ", i);
}
```
출력:

    1 2 4 5

## 중첩 반복문

- 반복문 안에 또 다른 반복문을 작성 가능
- 2차원 구조, 구구단, 별찍기 등에 자주 사용

ex)

```c
// 구구단
for (int i = 2; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        printf("%d x %d = %d\n", i, j, i * j);
    }
}
```
